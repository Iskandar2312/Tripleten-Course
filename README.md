# Data

- `userID` — ID pengguna
- `track` — judul lagu
- `artist` — nama artis
- `genre` — genre musik
- `city` — kota asal pengguna
- `time` — waktu saat lagu tersebut diputar
- `day` — hari dalam seminggu
